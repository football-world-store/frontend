# Documentação — Football World Store Frontend

Esta pasta contém a documentação técnica do projeto. Serve como contexto para desenvolvedores e agentes de IA.

## Índice

| Documento | Descrição |
|-----------|-----------|
| [ARCHITECTURE.md](./ARCHITECTURE.md) | Stack, estrutura de pastas, fluxo de dados, auth, API client |
| [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md) | Paleta de cores, tipografia, regras visuais, tokens, utilities |
| [DOMAIN.md](./DOMAIN.md) | Entidades, linguagem ubíqua, perfis, regras de negócio |
| [CONVENTIONS.md](./CONVENTIONS.md) | Nomenclatura, padrões de código, limites, proibições |
| [IMPLEMENTATION_STATUS.md](./IMPLEMENTATION_STATUS.md) | Status de implementação vs requisitos contratados |

## Para IAs

Se você é um agente de IA trabalhando neste projeto:

1. Leia `ARCHITECTURE.md` para entender a estrutura
2. Leia `DESIGN_SYSTEM.md` antes de tocar em qualquer componente visual
3. Leia `DOMAIN.md` para entender as entidades e regras de negócio
4. Leia `CONVENTIONS.md` para seguir os padrões do projeto
5. Consulte `IMPLEMENTATION_STATUS.md` para saber o que já existe e o que falta
